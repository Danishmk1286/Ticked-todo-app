## What is Ticked?

Ticked is a clean, distraction-free task manager that lives entirely in your browser. No sign-up, no backend, no clutter — just you and your tasks.

---

## Features

- Add tasks and mark them complete with an animated strikethrough
- Expandable notes panel per task for extra context
- Progress bar with motivational milestone messages
- Completed tasks auto-sort to the bottom, incomplete stay on top
- Drag and drop to reorder your list
- Everything persists in localStorage — survives refresh and browser close
- Delete individual tasks or reset everything, both with confirmation
- Fully responsive — works on mobile, tablet and desktop
- Zero dependencies, zero backend, zero tracking

---

## Tech

Built with pure HTML, CSS and vanilla JavaScript. Single `.html` file. No frameworks, no build step, no npm install.

## Usage

No installation needed. Either:

- 📁 Download `index.html` and open it in any browser
